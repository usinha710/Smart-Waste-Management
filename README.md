♻️ Smart Waste Management System

A web-based Smart Waste Management platform designed to improve communication between citizens and waste management authorities. The system allows users to report waste issues, while administrators can monitor complaints and manage waste collection more efficiently.

The goal of this project is to make waste management more organized, transparent, and efficient, helping cities maintain cleaner environments.

🚀 Features
👤 User Features

1.User registration and login

2.Submit waste complaints

3.Waste detection / prediction module

4.Contact and information pages

🛠️ Admin Features

1.Secure admin login

2.Admin dashboard to monitor complaints

3.View and manage waste reports submitted by users

4.Track complaint history

⚙️ Installation & Setup
1 Clone the repository
git clone https://github.com/usinha710/Smart-Waste-Management.git
2 Go to project folder
cd Smart-Waste-Management
3 Install dependencies
npm install
4 Run the server
node sign_in_backend/server.js
💡 How It Works

Users register or log in to the system.

Users can report waste issues through the platform.

Complaint data is stored in the database.

Admin accesses the dashboard to monitor and manage reports.

The system helps streamline waste management processes.
